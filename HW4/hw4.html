Part 1: Fit a Neural Field to a 2D Image

Model Architecture Report

  • Input: Normalized 2D pixel coordinates (x, y) ranging [0, 1].
  • Positional Encoding: Maximum frequency L=10, generating a 42-dimensional input feature vector.
  • MLP Backbone: 4 hidden layers. Each layer consists of Linear(256) → ReLU.
  • Output Layer: Linear(256, 3) → Sigmoid mapping features to (r, g, b) pixel colors clamped to [0, 1].
  • Optimizer: Adam optimizer with Learning Rate: 1e-2 (0.01).
  • Task: Mean Squared Error over batches of 10,000 random pixels to memorize a static frame.

Part 2: Fit a Neural Radiance Field from Multi-view Images

Pipeline Implementation Details

  • Camera Extrinsics & Ray Generation: Utilizing the provided c2w transforms and focal distances, physical rays are reverse-projected out of the sensor. The RaysData class precomputes millions of theoretical ray_o (Ray Origins) and ray_d (Ray Directions) into a unified PyTorch dataset pool for efficient 10,000-ray memory batching.
  • Volume Sampling: sample_along_rays slices every theoretical ray line into exactly 64 finite distance fragments spanning the mathematical bounding box (near=2.0, far=6.0). By intentionally applying jitter algorithms offset noise across distance batches during training (perturb=True), the model avoids discrete depth over-fitting and establishes true continuous volume approximations.
  • 3D NeRF Architecture: An 8-layer deep MLP mapping full 3D coordinates. To bypass the "Dead ReLU" catastrophe over massive coordinates, implicit PyTorch PI multipliers were omitted during positional encodings. Due to exponential memory loss scaling with architectural depth, original geometric xyz encoding bypass coordinates are manually concatenated at Layer 5 (Skip Connection).
  • View-Dependent Rendering: Density (σ) remains direction invariant. The viewing direction ray_d vector is withheld entirely until the final structural block, wherein it merges with semantic features to uniquely map viewport-specific physical attributes like plastic shading glares across RGB pixels.
