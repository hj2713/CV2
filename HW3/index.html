Overview

This project implements components of a simple Structure from Motion (SfM) pipeline that estimates correspondences, camera pose, and 3D scene points from a pair of images:

  • Step 1: Camera Intrinsics (Calculated from iPhone 15 Pro specifications)
  • Step 2: Detecting SIFT Features (rotation + scale invariant)
  • Step 3: Feature Matching using L2 distance and Nearest Neighbor Distance Ratio (NNDR)
  • Step 4: RANSAC to estimate Rotation (R) and Translation (t) via Essential Matrix decomposition
  • Step 5: Triangulating inliers to generate a 3D Sparse Point Cloud
